# 笼中凝视 · The Caged Gaze

你醒来时已经在封闭设施里。周围是11个陌生人。  
广播说，这是一场社会实验——你们之中，混入了3-5个能够完美模仿人类的“伪人”。  
每天早晨，广播会播报一则特殊事件——停电、警报、测谎仪、匿名纸条……这些事件会揭露每个人的可疑之处。你必须观察、记录、在他们撕下伪装前找出他们。  
投票、处决、活过7天。找出异常，或者死在错误的判断里。  
祝你好运。

## 关于

《笼中凝视》是一款纯文字交互心理惊悚游戏。没有画面，没有战斗系统，也没有传统意义上的探索地图——只有对话、投票、档案，以及不断累积的违和感。

你将被置入一场持续七天的封闭“信任实验”中，与11名身份不明的参与者共同生活。你需要通过日常对话、晨间广播、突发事件与投票环节，判断谁才是真正异常的那一个。

## 故事

你被告知，自己正参与一场封闭社群信任实验。

设施内共有12名参与者，其中混入了3到5个能够完美模仿人类的“伪人”。它们拥有正常的外表、流畅的语言、合理的记忆与稳定的情绪反馈。它们会微笑，会解释，会表现得比任何人都更像一个“正常人”。

而你的任务，是在七天内把它们找出来。

每天早晨，广播都会发布一则特殊事件。  
停电、警报、失窃、匿名纸条、规则变更、强制隔离——这些异常会一点点撕开所有人的伪装，也逼迫你重新理解“破绽”究竟意味着什么。毕竟，完美的才是机器，会恐惧的才是人类。

## 玩法

- 阅读每日对话，观察每个角色的反应与行为变化
- 根据晨间特殊事件提供的线索，记录可疑对象
- 在投票阶段指认你认为最可疑的人
- 活到第七天，并尽可能接近真相
- 解锁档案室内容，回看NPC档案、事件记录、结局记录与机密文件

## 系统特色

- 动态身份分配：每次实验重启，系统都会重新分配“观察者”与“被观察者”。同一角色在不同周目中，其行为逻辑与可疑破绽都会发生致命的变化
- 11个晨间特殊事件：每个事件都可能暴露不同类型的异常
- 多结局结构：包含失败结局、真结局与隐藏结局
- 档案收集系统：通关后可逐步解锁NPC档案、事件记录、结局记录与机密文件
- 多周目体验：随着理解加深，玩家会逐渐看到表象背后的另一层逻辑
